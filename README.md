# Overview

Companion resources for DISSC's AI Pair Programming workshops, designed to help you learn the tools and techniques for coding effectively with AI assistants.

## Workshop Tracks

### Beginner
For those new to AI-assisted development.

**This track is for you if:**
- You're new to software development or have worked primarily in R for data science tasks
- You haven't used VS Code or similar code editors
- Your AI experience is primarily through chat interfaces (e.g., ChatGPT, Claude)
- You're curious about AI coding tools but are unsure how/where to start

### Advanced
For experienced practitioners looking to level up.

**This track is for you if:**
- You already use AI coding tools (e.g., Copilot, Cursor, Claude Code) regularly
- You're comfortable with VS Code or similar editors
- You've integrated AI into your development workflow
- You want to share techniques and learn from other practitioners
