![Yale's DISSC AI Workshops](assets/header.svg)

Hands-on workshops from the [Data-Intensive Social Science Center (DISSC)](https://dissc.yale.edu) that teach researchers how to use AI tools in their daily work. Designed for faculty, pre- and postdoctoral researchers, staff, and graduate students across disciplines. No prior experience with AI tools or command-line interfaces required.

Participants work inside a secure, sandboxed Docker container with all tools pre-installed. Data never leaves Yale's AWS infrastructure and is never used for model training. The container restricts AI agent access to the project folder, preventing unintended changes elsewhere on your computer.

## AI Foundations

Best practices for AI-powered research workflows. Two 45-minute sessions with a 45-minute lunch break.

**AI Foundations I: Setup & Tools**
Install Docker, VS Code, and Claude Code. Understand how containers, terminals, and the AI service connect. Verify your environment is configured and ready to use.

**AI Foundations II: AI-Accelerated Research**
Use Claude Code to explore a research question, write and iterate on R code, build visualizations, and debug. Covers prompt writing, model selection, memory, custom skills, container safety, and best practices for AI-powered workflows.

Currently active with [VS Code + R](workshops/ai-foundations/vs-code-and-r/). Adaptable to Python and Stata. An earlier [Positron + R](workshops/ai-foundations/positron-and-r/) version is archived for reference.

## Getting Started

1. Click the green **Code** button at the top of this page, then click **Download ZIP**

3. Follow the setup instructions (AI Foundations I) and work through the exercises (AI Foundations II)
