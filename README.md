# Overview

Companion resources for DISSC's AI Pair Programming workshops, designed to help you learn the tools and techniques for coding effectively with AI assistants.

## Workshop Tracks

### Beginner
For those new to AI-assisted development. Whether you've seen demos of AI coding tools or are just getting started, this track covers the fundamentals of working effectively with AI assistants.

### Advanced
For experienced practitioners looking to level up. Share and learn tips, tricks, and techniques with others who are already integrating AI into their daily workflows.
